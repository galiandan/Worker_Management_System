# 职工管理系统 (Worker Management System)

本项目是一个基于 **C++ 多态** 特性实现的轻量级职工管理系统。通过封装、继承和多态三大特性，实现了对不同岗位职工的统一管理与数据持久化。

---

## 📖 项目简介
本教程主要利用 C++ 来实现一个基于多态的职工管理系统。系统将职工分为三类：**普通员工、经理、老板**。在显示信息时，会展示职工编号、职工姓名、职工岗位以及各自的职责。

### 👥 职位与职责描述
| 职位 | 职责内容 |
| :--- | :--- |
| **普通员工** | 完成经理交给的任务 |
| **经理** | 完成老板交给的任务，并下发任务给员工 |
| **老板** | 管理公司所有事务 |

---

## 🛠️ 系统功能模块
该管理系统涵盖了日常人力资源管理的核心功能：

* **1. 退出管理程序**：安全退出当前运行的管理系统。
* **2. 增加职工信息**：支持**批量添加**职工功能，并将信息录入到本地文件中。
    * *录入信息包含：职工编号、姓名、部门编号。*
* **3. 显示职工信息**：一键显示公司内部所有在职职工的详细资料。
* **4. 删除离职职工**：按照**职工编号**精准删除指定的职工记录。
* **5. 修改职工信息**：按照**职工编号**定位并修改职工的个人资料。
* **6. 查找职工信息**：支持按**职工编号**或**职工姓名**进行灵活检索。
* **7. 按照编号排序**：支持按职工编号进行排序，排序规则（升序/降序）由用户指定。
* **8. 清空所有文档**：一键清空文件中记录的所有职工信息。
    * ⚠️ *注：清空前设有再次确认环节，防止误操作。*

---

## 🚀 技术实现要点
* **多态性 (Polymorphism)**：通过抽象基类 `Worker` 派生出不同职位类，利用虚函数实现运行时绑定。
* **文件操作 (File I/O)**：利用 `fstream` 库实现数据的读取与保存，确保数据持久化。
* **动态内存管理**：使用指针数组动态维护职工对象，提高系统灵活性。
